Set up a self hosted Landscape Server on Ubuntu 18.04

  1. Add Landscape's repository

    sudo add-apt-repository --update ppa:landscape/19.10

  2. Install Landscape

    sudo apt install landscape-server-quickstart

For additional information, please reference the Landscape Quickstart deployment documentation ›
