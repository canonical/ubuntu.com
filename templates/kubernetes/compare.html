Secure Kubernetes

Hardened platform. Immutable containers. Security patching.

Canonical Kubernetes runs in immutable containers to provide for better security out of the box. Integration with K8s RBAC, Active Directory and LDAP, CIS hardening by default, encryption at rest and automatic security patching ensure you get the most secure Kubernetes in the market.
