Other distributions

MicroK8s

Lightweight, low-touch Kubernetes for cloud, edge and IoT. It brings all Kubernetes services and the most popular addons in a single package, and requires minimal ops for easy clustering, self-healing HA and simpler upgrades. Secure by default and CNCF conformant.

Read the MicroK8s documentation ›

Charmed Kubernetes

Composable, operator-based Kubernetes for the enterprise, with full life-cycle management for host and in-cluster with Juju. It provides carrier-grade and hardware acceleration features, and support for third-party components and services. It gives you fully customizable deployments, with pluggable CNI, CSI, CRI and monitoring components. Secure by default and CNCF conformant.
