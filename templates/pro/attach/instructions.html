Where can I find my Ubuntu Pro code?

{{ image( url="https://assets.ubuntu.com/v1/5baafba5-security_center_magic_attach_1.png", alt="", width="1704", height="1408", hi_def=True, loading="auto" ) | safe }}

Ubuntu Pro can be enabled from the Security Center app. On 24.04 LTS and older, use the Software & Updates app instead.

  1. Go to the "Ubuntu Pro" page and click on "Enable Ubuntu Pro".

  2. Select "Enable with Ubuntu Pro account".

  3. Continue in browser as instructed, or copy the six-character code and paste it in the field above on this page.

  4. Select the subscription you want to attach to, and submit.

{{ image( url="https://assets.ubuntu.com/v1/4119ff56-magic_attach_visual_aid_server.png", alt="", width="1174", height="615", hi_def=True, loading="auto" ) | safe }}

During installation, you'll be asked whether you want to Upgrade to Ubuntu Pro.

  1. Choose "Enable Ubuntu Pro".

  2. On the next screen you will be presented with a six-character code as part of the first option.

  3. Enter the code into the field above on this page, select the subscription you want to attach to, submit.
