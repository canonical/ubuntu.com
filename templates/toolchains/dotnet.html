Predictable .NET release cadence on Ubuntu

Our partnership with Microsoft means that Ubuntu releases follow an established, reliable cadence that offers you both the latest features of Ubuntu, and support for the newest versions of .NET shortly after release.

Here are the details:

  • Starting with Ubuntu 22.04, every new Ubuntu LTS release ships with the latest .NET release in the archive.
  • Each new interim Ubuntu release will include all supported .NET releases.
  • Additional backports are available thanks to the .NET backports PPA .
  • Upcoming release previews are available thanks to the .NET Previews PPA .

This makes it simple for developers to experiment with the latest versions as soon as possible, while having assurance that their production workload will be supported for a very long time.

Secure, minimal containers for .NET

Canonical provides chiseled containers for the .NET runtime environment, built in collaboration with Microsoft.

These containers provide a significant reduction in size and attack surface, resulting in better performance, lower resource requirement, and reduced security risks.

The images are available for AMD64, ARM64, S390X and PPC64EL architectures, and benefit from up to 12 years of security support with Ubuntu Pro.

Improved dev environment setup with the .NET snap

{{ image(url="https://assets.ubuntu.com/v1/9dfa2145-imrpoved-dev-env.png", alt="", width="1200", height="800", hi_def=True, loading="auto|lazy", fmt="png" ) | safe }}

The .NET snap simplifies the .NET installation process on Ubuntu. The snap enables the simultaneous installation of various .NET SDKs and runtimes, all using the same source-built binaries available from the Ubuntu archive.

Additional .NET content snaps are available for both the .NET and ASP.NET runtimes and SDK. They share .NET runtimes and SDK across multiple .NET application snaps, saving storage space and reducing installation times.

Simplified application packaging with Snapcraft plugin

The new .NET plugin enables you to automate, standardize and simplify the process of building snaps for .NET applications by letting Snapcraft handle all the language specific building logic.

Benefit from improved support for monorepo setups, it integrates better with MSBuild through custom flag attributes and it leverages the native .NET content snaps.
