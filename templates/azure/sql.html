What's inside SQL Server on Ubuntu Pro?

24/7 support

Enterprise-grade combined support available via your existing Microsoft support agreement.

Choose the right SQL Server for you

Four flavours of SQL Server 2025 are available – Enterprise, Standard, Web and Developer editions are ready to launch today.

FIPS & CC-EAL2 certified

Officially certified components to enable operating environments under compliance regimes like FedRAMP, HIPAA, PCI and ISO.

Fast and optimized

SQL Server on Ubuntu Pro is optimized for performance and stability, with out-of-the-box support for PMEM-enabled instances where available.

Ready for the unexpected

The solution includes a dedicated Azure high availability fencing agent, for better failover times and greatly improved reliability.

CIS and STIG

For companies looking to leverage industry benchmarks for hardening, Ubuntu Pro makes two leading implementation guides available.

Up to 15-year lifetime

Canonical backs SQL Server on Ubuntu Pro for up to 15 years, ensuring security updates are available throughout.

Retain governance

You decide how and when patches, updates and other lifecycle actions take place – for full compliance and control.

Reduce unplanned downtime

Ubuntu kernel updates are regularly issued. With our Livepatch service, kernel patches are delivered immediately, without the need to reboot your VMs.

Backed by Canonical and Microsoft.
