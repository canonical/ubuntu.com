What's in Ubuntu Pro?

FIPS & CC-EAL2 certified

Officially certified components to enable operating environments under compliance regimes like FedRAMP, HIPAA, PCI and ISO.

CIS and DISA STIG

For companies looking to leverage industry benchmarks for hardening, Ubuntu Pro makes two leading implementation guides available.

Open source security

Ubuntu Pro adds security coverage for the most important open source applications like Apache Kafka, NGINX, Redis and PostgreSQL.

Predictable and optimized

Ubuntu Pro is based on Ubuntu and leverages both the breadth of open source included and the Azure optimizations built in.

Kernel Livepatch

Ubuntu kernel updates are regularly issued. With our Livepatch service, kernel patches are delivered immediately, without the need to reboot your VMs.

Billed through the cloud

Choose between annual fixed or consumption-based pricing, with billing simplified through direct integration with your Azure bill and counting towards your Microsoft Azure Cloud Consumption (MACC) agreement.

Seamless integration with your Azure

Ubuntu Pro offers full integration with core Azure services such as Azure Pricing, Azure Guest Patching, and Update Management Center.

Up to 12 years lifetime

Canonical backs Ubuntu Pro for up to 12 years, ensuring security updates are available throughout, with a guaranteed upgrade path.

Optional 24/7 support

Additional enterprise-grade support is available through a private offer for Ubuntu Pro infrastructure and applications.
