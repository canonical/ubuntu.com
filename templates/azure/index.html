Ubuntu on Azure

Canonical produces official Ubuntu images on Microsoft Azure with continuous updates to ensure security and stability from the moment your machines and containers launch.

Microsoft works closely with Canonical to optimize Ubuntu images on Microsoft Azure and ensure Ubuntu supports the latest cloud features as these are released. Today, Ubuntu powers a large number of mission-critical workloads on Azure.

Beyond Ubuntu, Canonical also offers Charmed Apps on Microsoft Azure, enabling hybrid environments.

Secure Data at Run-Time with Confidential Computing on Azure

If you're running critical workloads and need to process sensitive data, security at system memory is essential. With confidential computing, your workloads are encrypted at run time, and are thus secure from all software vulnerabilities outside the VM perimeter – including the hypervisor and from unauthorized cloud admins. Pair confidential VMs with Ubuntu Pro for security in and out of your VM.

  • Secure during compute
  • Enables zero trust
  • Process sensitive workloads without worry
  • Available on Ubuntu Pro for added security or the free versions of Ubuntu
