Ubuntu solutions for the public sector,
delivered by Canonical

Ubuntu is a multi-cloud platform

Ubuntu provides a secure, certified platform across multi-cloud and hybrid cloud environments. We can work with you to build, deploy, operate and support your infrastructure - AWS, Azure, Google, VMware, Hyper-V, OpenStack and Docker and Kubernetes containers. We enable multi-cloud portability for the public sector.

Comprehensive open source support

Get open source support from a single vendor for all of the layers of your stack, from OS to applications. Canonical provides a unified model for open source support services with transparent per-node pricing, avoiding the often high and hidden fees that come with per-core or per-socket pricing models.

Secure, upstream OpenStack and K8s

Take advantage of best-in-class server virtualisation with Charmed OpenStack and production-grade container orchestration with Charmed Kubernetes on Ubuntu. Canonical provides consulting, design/build, training and 24/7 enterprise-grade support services for key open source technologies.

Enterprise-grade management tools

Canonical provides an automation portfolio with multi-tier service modelling. Offering metal-as-a-service (MAAS) for automated server provisioning, Juju for configuration and deployment, and Landscape for Ubuntu systems management at scale for audit and compliance reporting, across your infra, service and application tiers.

Platform across all of IoT, from cloud to edge

Ubuntu Core is a security-first platform built on the new containerised linux packaging format, Snaps, delivering stronger resiliency and security. From smart drones and autonomous vehicles to robots, Ubuntu is the new standard for embedded Linux. Get the world’s best security, a custom app store and reliable updates.

Classified, isolated or limited networks

Reduce time to deployment from one day to one hour. Rapidly deploy the latest in cloud technology to isolated, classified (high side) or slow, bandwidth-limited networks. Implementing Canonical’s AIRstack process provides a secure and compliant on-premise deployment solution.

Azure and AWS compliance with Ubuntu Pro

Ubuntu Pro and Ubuntu Pro FIPS are premium images that are optimised for Azure and AWS with compliance, security and hardening for production environments. Ubuntu Pro includes comprehensive, 10-year open source security coverage for key applications, like Apache Kafka, NGINX and PostgreSQL, access to kernel Livepatch service to avoid unplanned downtime and is priced for the cloud.

The Ubuntu Pro FIPS premium images provide access to AWS and Azure FIPS and CC-EAL2 certified components to enable operating environments under compliance regimes. Ubuntu Pro FIPS is built upon the power of Ubuntu Pro's enhanced stability and security features and provides a compliant platform for government and public sector agencies and contract vehicles.
