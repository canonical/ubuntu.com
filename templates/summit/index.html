Featured speakers

The Ubuntu Summit will include a series of talks, workshops, panels and Q&A. We cover a broad range of topics to appeal not only to developers, but anyone involved in open source. You will learn about Linux Desktop, cloud and infrastructure, community, arts, design, and much more.

Topics

{{ image(url="https://assets.ubuntu.com/v1/fed38b56-new-linux.png", alt="", width="566", height="380", hi_def=True, attrs={"class": "p-image-container__image u-zoom-on-hover"}, loading="lazy") | safe }}

Linux Desktop

Sessions on this track will focus on Linux desktop technologies and applications, desktop environments, Raspberry PI, packaging.

{{ image(url="https://assets.ubuntu.com/v1/ec34b48a-new-app-ecosystem.png", alt="", width="566", height="380", hi_def=True, attrs={"class": "p-image-container__image u-zoom-on-hover"}, loading="lazy") | safe }}

Application Ecosystem

This track will showcase Open Source applications, programming languages, APIs and tools to build and deploy applications.

{{ image(url="https://assets.ubuntu.com/v1/8a92eed1-new-content-design.png", alt="", width="566", height="380", hi_def=True, attrs={"class": "p-image-container__image u-zoom-on-hover"}, loading="lazy") | safe }}

Content and Design

This track will take you on a journey to discover what makes a great user experience on a wide range of products and using different technologies.

{{ image(url="https://assets.ubuntu.com/v1/c71d8fa6-new-community.png", alt="", width="566", height="380", hi_def=True, attrs={"class": "p-image-container__image u-zoom-on-hover"}, loading="lazy") | safe }}

Community

This track will focus on community building and management, tools and processes useful to community leaders, and documentation in open source projects.

{{ image(url="https://assets.ubuntu.com/v1/359e01a1-new-infrastructure.png", alt="", width="566", height="380", hi_def=True, attrs={"class": "p-image-container__image u-zoom-on-hover"}, loading="lazy") | safe }}

Infrastructure

This track is focused on Linux server, infrastructure and cloud: where and how they can be deployed, and how they are optimized on platforms ranging from raspberryPI to IoT.

{{ image(url="https://assets.ubuntu.com/v1/5fba2c50-new-devices.png", alt="", width="566", height="380", hi_def=True, attrs={"class": "p-image-container__image u-zoom-on-hover"}, loading="lazy") | safe }}

Devices

From small smart home devices to robots in space stations, Linux is everywhere. This track celebrates our community work in the robotics and IoT field.

{{ image(url="https://assets.ubuntu.com/v1/70b56649-new-data.png", alt="", width="566", height="380", hi_def=True, attrs={"class": "p-image-container__image u-zoom-on-hover"}, loading="lazy") | safe }}

Data, MLOps and AI/ML

This landscape is a dynamic one, with new tools, frameworks and models emerging daily. This track is focused on Data, MLOps and AI/ML and how open source empowers the next technological revolution.

{{ image(url="https://assets.ubuntu.com/v1/5c5e4abc-new-gaming.png", alt="", width="566", height="380", hi_def=True, attrs={"class": "p-image-container__image u-zoom-on-hover"}, loading="lazy") | safe }}

Gaming

In this track, you'll learn about the people behind the gaming software, creative tools and and APIs that are powering this unstoppable phenomenon.

{{ image(url="https://assets.ubuntu.com/v1/e298d421-new-security.png", alt="", width="566", height="380", hi_def=True, attrs={"class": "p-image-container__image u-zoom-on-hover"}, loading="lazy") | safe }}

Security

Security is fundamental to Ubuntu, Canonical, and the open source community. This track will focus on the latest developments and practices to keep users and information secure.
