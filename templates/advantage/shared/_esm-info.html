ESM is active by default on any newly-attached Ubuntu 14.04 ESM or 16.04 ESM system.

Ubuntu 14.04 ESM and 16.04 ESM
To activate ESM Infra sudo ua enable esm-infra
To deactivate sudo ua disable esm-infra

ESM Infra is not required for 18.04 LTS and 20.04 LTS as they are currently within the 5 years free LTS maintenance period. Learn more about the Ubuntu LTS release cycle.
