• What Ubuntu version are you running?

    Ubuntu Advantage is available for Ubuntu 14.04 and higher.
    Are you using an older version?

    For Ubuntu 18.04, all UA plans include:

    • Kernel Livepatch to avoid unscheduled reboots
    • Ubuntu systems management with Landscape
    • Knowledge base access
    • Certified Windows drivers for KVM guests
    • FIPS 140-2 Level 1 certified crypto modules
    • CIS benchmark and automation tooling
    • DISA STIG
    • Extended Security Maintenance (ESM) until 2028

    • Other versions?

    Ubuntu Advantage is available only for Ubuntu 20.04 LTS, 18.04 LTS, 16.04 LTS, and 14.04 ESM.
