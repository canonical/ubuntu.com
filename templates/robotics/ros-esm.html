What is ROS ESM?

Whether you want to be compliant with cybersecurity regulations, win more time before your migration, or reduce operational expenses of maintaining upstream repositories, ROS ESM is here to make your work easier.

Robot Operating System Expanded Security Maintenance (ROS ESM) is a service by Canonical that provides security maintenance for ROS Long Term Support (LTS) releases and the underlying Ubuntu distributions, starting with ROS Kinetic.

This includes the underlying Ubuntu system and more than 23,000 packages from the Ubuntu Universe repository.

ROS ESM is available with your Ubuntu Pro subscription.

Security maintenance you can trust

Running unpatched and unmaintained versions of ROS exposes your robot, company and customers to serious risk. The challenge increases when working with End-Of-Life (EOL) ROS distributions or dependencies such as Python 2.

With ROS ESM, you can count on backports for critical security updates, common vulnerabilities and exposures (CVE) fixes and bug fixes for ROS and Ubuntu base OS distribution. Our security experts follow a standardised process to identify vulnerabilities and create and test security patches.

With ROS ESM, the time-consuming and resource-intensive work of keeping core ROS packages secure is no longer a problem.
