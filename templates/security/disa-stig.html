What is DISA-STIG?

The Defense Information System Agency (DISA) is a US Department of Defense combat support agency. It provides and operates information infrastructure to support military operations and national-level leadership. The Security Technical Implementation Guide (STIG) is a configuration standard consisting of guidelines for hardening systems to improve a systemâ€™s security posture. It can be seen as a checklist for securing protocols, services, or servers to improve the overall security by reducing the attack surface.

How to audit and comply with DISA-STIG?

  1. Auditing

    Using the Ubuntu Security Guide auditing is as simple as:

    sudo usg audit disa_stig

    and get the report in machine readable or browser friendly format.

  2. Compliance

    Using the Ubuntu Security Guide applying the necessary rules for compliance is as simple as:

    sudo usg fix disa_stig
