What is DISA-STIG?

The Defense Information System Agency (DISA) is a US Department of Defense combat support agency. It provides and operates information infrastructure to support military operations and national-level leadership. The Security Technical Implementation Guide (STIG) is a configuration standard consisting of cybersecurity requirements for a product such as Ubuntu. It can be seen as a checklist for securing protocols, services, or servers to improve the overall security by reducing the attack surface.

The DISA-STIG guides for Ubuntu
