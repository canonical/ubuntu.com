Details

  • wrapper_template - this needs to be "engage/_base_engage_markdown.html"
  • title - page title for the document head
  • meta_description - meta_description for the document head
  • meta_image: url of the image for social media meta data
  • meta_copydoc: - url of the google copy doc
  • header_title - text for the <h1> of the pages
  • header_subtitle - optional - text for the <h4> subtitle
  • header_image: - optional - url of the image to be on the right of the banner
  • header_image_width - required the width of the header image in pixels
  • header_image_height required - the height of the header image in pixels
  • header_url - the url or anchor for the mobile view CTA - usually '#register-section'
  • header_cta - text for the mobile view CTA - usually 'Register for the webinar' or 'Download the whitepaper'
  • header_class - the class of the engage banner you want to use - usually one of: p-engage-banner--dark, p-engage-banner--grad, p-engage-banner--k8s, p-engage-banner--openstack, p-engage-banner--aqua, p-engage-banner--snapcraft
  • header_lang - optional - do not add for english
  • webinar_code: - optional - the full html of the webinar embed code from brighttalk
  • form_include - optional - the language code of the form to use - either 'en, 'de', or 'fr'
  • form_id - optional - the marketo form id
  • form_return_url - optional - the url you want to send users to after they fill in the form, usually a marketo thank-you page
