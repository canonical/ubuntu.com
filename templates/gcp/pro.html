Let's talk Pro

Broadest open source security coverage keeps CISOs sleeping at night

Ubuntu Pro includes security coverage for the entire collection of software packages shipped with Ubuntu, tracking high and critical CVEs. This makes it ideal for companies that have embraced open source for their new products and need to ensure security for production and mission-critical workloads.

One key reason why developers prefer Ubuntu is that it includes a huge set — in total, over 30,000 packages — of upstream open source known as ‘universe’. Universe includes workloads like NGINX, MongoDB, Redis, Kafka, but also language ecosystems like Python, Java, Ruby and JavaScript/Node.js. With Ubuntu Pro's built-in CVE tracking and patching, CIOs and security officers can be confident they can roll these packages out to production.
