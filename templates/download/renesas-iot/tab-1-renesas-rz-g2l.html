Renesas RZ/G2L

Ubuntu Desktop 24.04

This is a reference image of Ubuntu on Renesas RZ/G2L IoT Platform for development and evaluation purposes.

Please note that, due to memory requirements of Ubuntu Desktop (minimum 4 GB RAM for 24.04), RZ/G2L evaluation kit is not Ubuntu Desktop Certified.

For production deployments, Ubuntu Server or Ubuntu Core images are recommended. Please contact Canonical for Ubuntu Desktop deployment on higher-memory RZ/G2L platforms.
