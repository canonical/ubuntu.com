Install Ubuntu Core on a Raspberry Pi

Install Ubuntu Core

We will walk you through the steps of flashing Ubuntu Core on a Raspberry Pi 2, 3, 4 or CM4. At the end of this process, you will have a board ready for production or testing snaps.

Minimum requirements

  • A Raspberry Pi 2, 3, 4 or CM3
  • A microSD card
  • An Ubuntu Core image
  • A monitor with an HDMI interface
  • An HDMI cable
  • A USB keyboard
