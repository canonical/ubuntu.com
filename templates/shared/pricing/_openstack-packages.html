  • Hardware guidance and sizing
  • Reference architecture

Fixed price design and deployment of a carrier-grade, feature-rich OpenStack cloud based on a custom architecture.

What's included: *

Everything in Private Cloud Build, plus:

  • Custom architecture

  • LDAP or Active Directory integration
  • Regulatory compliance
  • Encryption in flight and at rest
  • EPA tuning options
  • Block, object and file storage
  • Layer–7 load balancer
